# HTTP.bob

Visual gallery of HTTP status codes with animated GIFs.

## Tech Stack

- Vanilla JavaScript (ESM)
- WebComponents (Custom Elements API)
- servor (dev server)

## Setup

```bash
pnpm install
```

## Development

```bash
pnpm run dev
```

## Project Structure

```
├── components/      # WebComponents
├── css/             # Global styles
├── public/          # Static assets (GIFs)
├── index.html       # Entry point
├── main.js          # App initialization
├── coming-soon.html
├── package.json
└── pnpm-lock.yaml
```

## Features

Displays HTTP status codes (100-504) with animated GIFs and their corresponding reason phrases.
